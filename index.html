Reclaim your bookmarks

Uncentralized

Your bookmarks are yours, and can stay yours. Make them available from everywhere, for your eyes only and share them only if you want.

Free and open by design

Designed as an open protocol, you can use whatever implementation that suits you the most.

Open formats

Import and export your bookmarks in open and widespread formats.

Care about your privacy

Privacy is the main goal here. We cipher all communications

How it works

Over HTTP

Herisson nodes works over HTTP, a solid and established protocol and use HTTP codes to communicates with other nodes.

Based on JSON

All the protocol exchanges are based on the well known JSON format, which implementation is widely spread.

Ciphered

Everything is already ciphered with asymetric cryptography. You don't need an extra HTTPS security layer. But you can if you feel paranoid ;-)

Independency

Use whatever implementations, it should work with other nodes. But you can use it alone, on an intranet or for any purpose at your convenience.

Getting started

As of now, the only implementation of the Herisson Protocol is the Herisson Wordpress plugin, this is how you can try it:

  1. Download the Herisson for Wordpress
  2. Install the extension
  3. Retrieve your bookmarks from your favorite place Firefox/Delicious
  4. Import them into your Herisson node
  5. Manage your bookmarks, save screenshots, etc
  6. Enjoy!
